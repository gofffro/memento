# memento
Класс Originator хранит состояние (строку).

Метод SaveState() создаёт объект Memento, в котором сохраняется текущее состояние.

Метод RestoreState() восстанавливает состояние из объекта Memento.

В Main сначала устанавливаем состояние, сохраняем его, меняем, а потом восстанавливаем сохранённое.

Паттерн Memento позволяет сохранить внутреннее состояние объекта и вернуть его позже, не раскрывая детали реализации.
